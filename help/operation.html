Operation

To remove a pair of tiles, click on a tile to select it, and then click on a matching tile to remove the pair. Both tiles must be open. If the tiles do not match, the first tile will be deselected and the second will be selected. Selected tiles are colored blue.

Right click a tile to select a tile and find other tiles that match it. The matching tiles will be colored green and can be selected as above.


Operation Keystroke
Undo the previous operation Ctrl+Z
Redo the most recently undone operation Ctrl+Shift+Z
Find a pair of matching tiles Ctrl+F
Review removed tiles Ctrl+H
Start a new game Ctrl+N
Restart the current game Ctrl+R
Start a numbered game F3
Toggle tournament mode (starts a new game) Ctrl+T
Toggle sound effects (on by default) F7
Show this help window F1
Show the rules of the game Shift+F1
Quit the game Ctrl+Q
